# 我的电子书

这是一个使用 MkDocs 创建的电子书项目。

## 本地开发

1. 克隆仓库
```bash
git clone https://github.com/qiuzefan/ebook.git
cd ebook
```

2. 安装依赖
```bash
pip install mkdocs
```

3. 本地预览
```bash
mkdocs serve
```

4. 构建
```bash
mkdocs build
```
